# Trading Service

This is a standalone service that runs the Alpaca trading algorithm with WebSocket connectivity.

## Local Development

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```bash
cp .env.example .env
# Edit .env with your actual keys
```

3. Run the service:
```bash
npm start
```
